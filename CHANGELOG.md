### Changed

## [1.0] - 2021-06-25

### Added

- Add gameplay features
  - Player character
  - Obstacles (with custom collision handling)
  - Patrolling guards (with raycasting-based field of view)
  - Victory and failure conditions
- Three basic levels
- Main menu and basic GUI
- Level editor
  - Save and load levels from files
